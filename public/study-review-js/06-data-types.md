# 데이터 타입

## 7가지 기본 데이터 타입

JavaScript에는 총 **7가지 기본 데이터 타입(자료형)**이 있습니다.

이러한 데이터 타입을 **"기본(Primitives, 원시) 데이터 타입"**이라고 합니다.

|     | 타입       | 영문      | 설명             |
| --- | ---------- | --------- | ---------------- |
| 1   | 문자열     | String    | 텍스트 데이터    |
| 2   | 숫자       | Number    | 숫자 데이터      |
| 3   | 불리언     | Boolean   | 참/거짓 값       |
| 4   | 널         | Null      | 의도적인 빈 값   |
| 5   | 언디파인드 | Undefined | 정의되지 않은 값 |
| 6   | 심볼       | Symbol    | 고유한 식별자    |
| 7   | 빅인트     | BigInt    | 매우 큰 정수     |

## String(문자열)

문자는 작은 따옴표, 큰 따옴표, 역 따옴표로 사용합니다.

```jsx
const 작은_따옴표 = '작은 따옴표';

const 큰_따옴표 = '큰 따옴표';

const 역따옴표 = `역따옴표`;
```

문자 안에는 이스케이프 문자를 사용할 수 있습니다.

```jsx
'줄바꿈\n문자';

'탭\t문자';

'백슬래시\\문자';

'작은따옴표\'문자';

"큰따옴표\"문자";
```

| 이스케이프 문자 | 의미              | 예시         | 출력       |
| --------------- | ----------------- | ------------ | ---------- |
| `\n`            | 줄바꿈 (New Line) | `'A\nB'`     | A          |
| B               |
| `\t`            | 탭 (Tab)          | `'A\tB'`     | A    B     |
| `\\`            | 백슬래시          | `'A\\B'`     | A\B        |
| `\'`            | 작은따옴표        | `'It\'s'`    | It's       |
| `\"`            | 큰따옴표          | `"\"안녕\""` | "안녕"     |
| `\r`            | 캐리지 리턴       | `'A\rB'`     | B (덮어씀) |
| `\b`            | 백스페이스        | `'AB\bC'`    | AC         |
| `\f`            | 폼 피드           | `'A\fB'`     | A          |
| B               |
| `\v`            | 수직 탭           | `'A\vB'`     | A          |
| B               |

## Number(숫자)

JavaScript에서는 숫자를 숫자 형태로 직접 작성하여 숫자를 만들 수 있습니다.

```jsx
// 정수
console.log(92_034); //가독성을 위해 천단위마다 언더스코어_ 사용 가능

// 실수 (정수 + 소수)
console.log(823 + 0.73);

// 음수
console.log(-100);

// 과학지수 표기법
1e5; // 100000
2.5e3; // 2500

// 2.5 × 10^6 = 2,500,000
console.log(25e5); // 25 × 10^5 ->10의 5 거듭 제곱 (10 * 10 * 10 * 10 * 10 = 100000)
console.log(2.5e6); // 2.5 × 10^6

// 특수 숫자 값
// * Infinity
// * -Infinity (-1 * Infinity)
// * NaN (Not a Number)

//JavaScript에서 안전하게 사용 가능한 큰/작은 정수
// * NUMBER.MAX_SAFE_INTEGER
// * NUMBER.MIN_SAFE_INTEGER
console.log(Number.MAX_SAFE_INTEGER); //9007199254740991
console.log(Number.MIN_SAFE_INTEGER); //-9007199254740991
```

<aside>
💡 특수 숫자 값

계산이 불가능하거나 정상적인 범위를 벗어났을 때 나타나는 일종의 **'비상 신호'**

### **1. Infinity (양의 무한대)**

어떤 숫자를 **0으로 나누거나**, 자바스크립트가 표현할 수 있는 **가장 큰 숫자(1.79...10^308)를 초과**할 때 발생합니다.

- **의미:** "사람이 셀 수 있는 범위를 넘어선 엄청나게 큰 값"
- **특징:** 무한대에 무엇을 더하거나 곱해도 여전히 무한대입니다.

### 2. -Infinity (음의 무한대)

양의 무한대와 반대로, 음수 방향으로 한계를 넘어섰을 때 발생합니다.

- **의미:** "음수 쪽으로 엄청나게 작은(절댓값은 큰) 값"
- **만드는 법:** `1`을 `0`으로 나누거나, `Infinity`에 `1`을 곱하면 됩니다.

### **3. NaN (Not a Number)**

가장 많이 마주하게 될 중요한 값입니다. 이름은 "숫자가 아니다"이지만, **아이러니하게도 타입은 숫자`Number` **입니다.

- **의미:** "계산을 시도했으나, 도저히 숫자로 결과를 낼 수 없는 **부적절한 연산**"

**언제 발생하나요?**

- 숫자가 아닌 것을 숫자로 바꾸려 할 때 (예: `'안녕' / 2`)
- `undefined`에 숫자를 더할 때
- `NaN`이 포함된 계산을 할 때 (결과도 무조건 `NaN`이 됨)
</aside>

<aside>
💡 **💎 JavaScript 안전한 정수 범위 (Safe Integer)**

자바스크립트의 숫자 타입`Number`은 내부적으로 **64비트 부동소수점(IEEE 754)** 방식을 사용합니다. 이로 인해 정수를 정확하게 계산할 수 있는 '안전 지대'가 정해져 있습니다.

**1. Number.MAX_SAFE_INTEGER**
자바스크립트에서 안전하게 사용할 수 있는 **최대 정수**입니다.
• **값:** `9,007,199,254,740,991` (약 9,007조)
• **의미:** 이 값보다 큰 정수는 자바스크립트가 정확하게 표현하지 못하고 다른 숫자와 동일하게 취급하거나 반올림할 수 있습니다.
• **공식 정의:** 2^53 - 1

**2. Number.MIN_SAFE_INTEGER**
자바스크립트에서 안전하게 사용할 수 있는 **최소 정수**입니다.
• **값:** `-9,007,199,254,740,991`
• **의미:** 이 값보다 작은(절댓값이 더 큰 음수) 정수는 정밀도를 잃게 됩니다.
• **공식 정의:** -(2^53 - 1)

**🔍 왜 "안전(Safe)"이라는 표현을 쓰나요?**
이 범위 내의 정수들은 **1대1로 정확하게 표현**되며, 산술 연산 (`+`, `-` 등) 시 오차가 발생하지 않음을 보장하기 때문입니다.

→범위를 벗어나면 컴퓨터가 숫자를 '어림짐작'하기 시작하여 논리적 오류가 발생합니다.

</aside>

## Undefined

`undefined` 값을 직역하면 "정의되지 않았다"입니다.

```smalltalk
// 선언만 하고 값을 할당하지 않음
// 1. 선언 ✅
// 2. 초기화: 의도적으로 값을 할당하지 않음 (엔진이 초기값으로 undefined 할당)
```

## Null

`null` 값을 직역하면 "비어있다"는 의미입니다.

개발자가 의도를 가지고 일부러 비워둔 값

```smalltalk
//예시:
// * 로그인 사용자
let loginUser = null // 로그인된 사용자가 없다. (의도를 가지고 비움)
```

# 참조형 데이터 타입

기본 데이터 타입 외에도 JavaScript에는 **참조형 데이터 타입(Reference Types)**이 있습니다.

참조형 데이터 타입은 **복잡한 데이터를 저장하고 조작**할 수 있는 강력한 기능을 제공합니다.

- 함수 (Function)
- 객체 (Object)
- 배열 (Array)

### Function

함수는 **특정 작업을 수행하는 코드 블록**을 말합니다.
요리 레시피처럼 특정 과정을 거쳐 결과를 만들어내는 것과 유사합니다.

```jsx
// 함수 (기능, 작업 정의)
function 커피_만들기(원두, 물, 얼음) {
  // 절차
  // 1. 원두 갈기
  console.log('원두 갈기');
  // 2. 물 끓이기
  console.log('물 끓이기');
  // 3. 커피 추출
  console.log('커피 추출');
  // 4. 커피 제조
  console.log('커피 제조');
  // 5. 완성된 커피 내보내기
  console.log('완성된 커피 내보내기');
}
```

### Object

객체 데이터 타입은 **여러 데이터를 하나로 묶어서 관리**합니다.

**객체(`{}`):** '커피\_메뉴'라는 커다란 대상의 **속성**(이름, 가격, 용량 등)을 정의합니다.

속성은 임의로 정할 수 있어서 자율도가 높습니다. (css는 정해져 있음)

객체 안에 배열이 들어 갈 수 있습니다.

```smalltalk
const 커피_메뉴 = {
  이름: '아인슈페너', // 변수, 값
  가격: 6_700, // 데이터는 콤마(,)로 구분
  용량: 'Large',
  핫음료: true,
  재료: ['에스프레소', '물', '크림'],
  만든이: '야무'
}
```

### Array

배열은 **여러 값을 순서대로 저장해 관리**합니다.

**배열(`[]`):** 속성 중 하나가 **여러 개의 하위 항목**을 가질 때 사용합니다.

```jsx
const 장바구니 = ['우유', '김', '치즈', '라면', '빵', '아이스크림'];
```
