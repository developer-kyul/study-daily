## 1. 실행 컨텍스트 (Execution Context)

실행 컨텍스트는 자바스크립트 엔진이 코드를 실행하기 위해 필요한 **모든 환경 정보를 모아놓은 객체**입니다.

### 💡 핵심 역할 및 종류

- **역할:** 변수/함수 저장, `this` 결정, 코드 순차 실행 관리.
- **전역 컨텍스트:** 파일 로드 시 생성, 프로그램 종료 시 소멸.
- **지역 컨텍스트:** 함수 호출 시 생성, 함수 종료 시 소멸.

### 🏗️ 구조 및 작동 원리

1.  **환경 레코드 (Environment Record):** 변수와 함수 선언을 저장하는 **재료 창고**. (호이스팅의 원인)
2.  **실행 스레드 (Thread of Execution):** 코드를 한 줄씩 실행하는 **조리대**. (싱글 스레드 방식)
3.  **작동 순서:** `생성 단계(선언부 등록)` ➔ `실행 단계(값 할당 및 실행)`.

## 2. 객체 (Object)

객체는 **키(key)-값(value)** 쌍으로 이루어진 데이터 꾸러미입니다.

### 🛠️ 기본 문법

- **생성:** 중괄호 `{ }`를 사용하며, 각 쌍은 쉼표 `,`로 구분합니다.
- **포함 가능 값:** 문자열, 숫자, 불린, 배열, 함수, 심지어 다른 객체까지 모든 값이 가능합니다.

### 🔍 속성(Property) 접근 및 조작

| 작업     | 점 표기법 (Dot Notation) | 대괄호 표기법 (Bracket Notation) |
| :------- | :----------------------- | :------------------------------- |
| **읽기** | `obj.key`                | `obj['key']`                     |
| **쓰기** | `obj.key = value`        | `obj['key'] = value`             |
| **삭제** | `delete obj.key`         | `delete obj['key']`              |

> **📌 대괄호 표기법을 쓰는 경우:**
>
> 1. 키에 **공백이나 특수문자**가 포함되어 있을 때
> 2. 키를 **변수**로 동적으로 받아올 때

## 3. 함수와 일급 객체 (First-class Object)

자바스크립트에서 함수는 단순한 동작이 아니라 **객체**로 취급됩니다.

### 🏃 메서드 (Method)

- **정의:** 객체의 속성으로 포함된 함수를 의미합니다.
- **호출:** `object.method()` 형태로 실행합니다.
