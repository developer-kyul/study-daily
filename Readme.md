# 주차별 문제 풀이

# 3주차 과제

### ✨과제 수행 과정 기록

매주 차 정리하는 내용을 포트폴리오로 활용할 수 있도록 컴퓨터 기반으로 정리하였다.
그래서 이번 주에는 배포까지 완료하여 이후에 과정에는 확장 및 수정을 하고 싶었다.  
 하지만 로컬 환경에서는 문제없이 잘 동작했지만, 배포 과정에서 예상치 못한 오류가 계속 발생했다.

원인을 찾지 못해 여러 시도를 반복했고, 일요일까지 해결되지 않으면 다른 플랫폼(벨로그 등)을 활용해야 하나 고민하기도 했다. 하지만 배포 방식 자체가 아니라 경로 설정 문제라는 것을 발견하고 수정하자, 그동안의 문제가 거짓말처럼 바로 해결되었다.

배포 환경과 로컬 환경의 차이를 몸소 체감할 수 있는 값진 경험이라고 생각한다. 😂

### ✨테스트 코드 작성

테스트 코드 작성 과제는 처음에는 다소 막연하게 느껴졌지만, 자세한 설명의 예시 코드가 제공되어 이해하는 데 큰 도움이 되었다. 단순히 코드를 작성하는 것이 아니라, **이 코드가 어떤 상황에서 어떻게 동작해야 하는가**를 먼저 생각하면서 만들 수 있어서 도움이 많이 되었다.

### ✨과제 코드 리뷰

과제 코드 리뷰는 이번 주차 과제 중 가장 “이제 정말 시작이다”라는 느낌을 강하게 준 과정이었다. 그동안의 학습 과정은 개념과 문법 위주의 이론 학습이 중심이었고, 주어진 내용을 이해하고 따라가는 데 집중하는 단계였다면, 이번 과제에서는 실제로 내가 어떤 구조로 코드를 작성해야 하는지 스스로 고민하게 되었다. 단순히 정답을 구현하는 것이 아니라, 코드를 어떻게 읽기 쉽게 구성할지, 다른 사람이 보았을 때도 이해할 수 있는지까지 생각하게 된 점이 인상 깊었다.

이 과정을 거치면서 단순한 학습용 과제를 넘어, 앞으로 내가 직접 만들게 될 웹페이지들과 실제 개발 과정이 자연스럽게 떠올랐고, 개발자로서 성장해 나갈 이후의 과정이 한층 더 기대되기 시작했다.

### ✨회고 (깨달은 점, 아쉬운 점 등)

이번 과제를 통해 배포까지 완료했다는 점은 분명 만족스러웠지만, 동시에 아쉬움도 크게 남았다. 기능 구현과 문제 해결에 집중하다 보니 코드 구조가 정리되지 않은 상태로 남아 있다는 점이 가장 크게 느껴졌다.

동작은 하지만 가독성이 떨어지고, 나중에 다시 보았을 때 한 번에 이해하기 어려운 코드들이 많았다. 매주차 정리하면서 코드를 더 깔끔하게 정리하면서, 배운 과정을 정리해야겠다.
